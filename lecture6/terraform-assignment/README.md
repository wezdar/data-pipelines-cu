# Terraform Assignment: Your First Webserver (Local)

Deploy a simple HTML page to a web server **locally** using Terraform and Docker, and retrieve the URL.

## Objectives

1. **Install Terraform** on your machine
2. **Create your first webserver** (Docker container running nginx)
3. **Deploy a simple HTML page**
4. **Retrieve the link** – output the URL to access your page (http://localhost:8080)

---

## Prerequisites

- **Docker** – [Install Docker](https://docs.docker.com/get-docker/) and ensure Docker daemon is running
- No cloud account needed – runs entirely on your machine

---

## Step 1: Install Terraform

### macOS (Homebrew)

```bash
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```

### Linux

```bash
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update && sudo apt install terraform
```

### Windows

Download from [terraform.io/downloads](https://www.terraform.io/downloads) or use Chocolatey:

```powershell
choco install terraform
```

### Verify Installation

```bash
terraform version
```

---

## Step 2: Create Your Webserver

1. **Clone or copy** the assignment files into a folder:

```
terraform-assignment/
├── main.tf      # Terraform configuration
└── README.md    # This file
```

2. **Initialize Terraform**:

```bash
cd terraform-assignment
terraform init
```

3. **Apply the configuration** (creates Docker container):

```bash
terraform apply
```

Type `yes` when prompted. This will:
- Create an `html/index.html` file
- Pull the nginx:alpine Docker image
- Run a container serving your HTML on port 8080

---

## Step 3: Retrieve the Link

After `terraform apply` completes, Terraform prints outputs:

```
Outputs:

local_url      = "http://localhost:8080"
container_name = "terraform-assignment-webserver"
```

**Open http://localhost:8080 in your browser** to see your deployed HTML page.

To view outputs again later:

```bash
terraform output local_url
```

---

## Step 4: Clean Up

Stop and remove the container:

```bash
terraform destroy
```

Type `yes` when prompted.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| `terraform: command not found` | Add Terraform to your PATH |
| Docker provider error | Ensure Docker is running (`docker ps`) |
| Port 8080 in use | Change the `external` port in main.tf or stop the process using 8080 |
| Cannot connect to URL | Wait a few seconds after apply for the container to start |
